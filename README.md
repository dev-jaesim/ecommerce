# E Commerce Website to practise ReactJS

A E commerce website developed by using ReactJS

---

## This project is using

- ReactJS
- Bootstrap
- React Router Dom
- React Icons
- React Context
- Styled Components
- Netlify

---

## Features

- [x] Detail page
- [x] Cart page
- [x] Adding, removing, updating product
- [x] Clear cart
- [x] 404 page
